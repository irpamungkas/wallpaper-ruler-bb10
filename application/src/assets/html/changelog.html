Change Log

Version 1.0.1

Released on: 05 September 2013

Features:

  • Scans for images on folders: Document, Camera, Photos & Downloads
  • Scale, rotate and move your images with touch gestures any way you want
  • Overlays: home screen, locked phone & active frames
  • The conext menu shows the file name on the title and the parent folder name on the subtitle of a selected image
  • Set edited image as device wallpaper or save it in folder Photos/Wappy
  • Once you select an image, use long press for the context menu
  • Use the device's cameras to take pictures from Wappy. Click the Camera button on the Gallery.
  • Share our application and spread Wappy to the world
  • Keep track of changes made to Wappy on the Changelog screen
